**Descripción**

DuckApp pro es un software de simulacion de carrera de patos, en el cual permite agregar carreras con sus respectivos nombres y participantes, mostrando un apartado visual de la simulación de la carrera de patos con el nombre de los participantes registrados, mostrar los tiempos, el ganador, primer segundo y tercer lugar, y las estadisticas de las carreras simuladas.

**Menu principal**
<img width="1037" height="768" alt="image" src="https://github.com/user-attachments/assets/134dcca5-a63e-4885-9cb2-1b652bca2ade" />

En el menu principal se muestra el apartado para acceder a todas y cada una de las secciones del software ya sea para agregar participantes, carreras, simularlas o ver las estadisticas.

**Carreras**
<img width="981" height="738" alt="image" src="https://github.com/user-attachments/assets/702cecf6-eef6-47a2-9d4c-83bfcbe9ca54" />

Este apartado muestra las carreras creadas y como crear las mismas. Para crear una carrera se ingresa el nombre de la carrera en el campo de arriba, se selecciona la caregoria y la fecha de la misma, para poder simular una carrera debe tener participantes que se agregan presionando la carrera que se quiere simular, se presiona el boton "agregar participante" para que a continucación aparezca la siguiente ventana. 
<img width="528" height="360" alt="image" src="https://github.com/user-attachments/assets/134d08e6-8c52-4a9e-ab4a-95f2a0bedb85" />
En esta ventana se muestran los participantes registrados se selecciona uno y se presiona "ok", posteriormente para simular la carrera se selecciona la carrera qué se desea y se selecciona "simular carrera" con esto la carrera se simula, muestra ganador, segundo y tercer lugar, además de la opcion de exportar el .txt. 
<img width="983" height="733" alt="image" src="https://github.com/user-attachments/assets/12dc0c8d-0abe-4de8-a1a4-b85b0aa0c588" />

**Simulación**
Para este apartado es la misma formula que de carreras solamente cambiando la ventana. Se selecciona la carrera, se preiona el boton "volver" y posteriormente el boton simulación, en el cual se abre la ventana de simulación, la cual permite, iniciar la carrera, pausarla, reiniciar y volver al menú anterior, al finalizar la carrera, muestra un mensaje de quien fue el ganador.
